# Herbal-Medicine-Predictor-Backend

## A Django project for Ayurvedic herbal medicine prediction using machine learning involve the following components:

  1. **Machine learning algorithms:** DecisionTreeClassifier, RandomForestClassifier, KNNClassifier, NaiveBayesClassifier, LogisticRegression

  2. **User interface:** The Django project incorprates **Angular** Framework for Frontend which is connected using **HttpClientModule**

  3. **Libraries Used**: <br>
     - **Joblib** - for saving Machine Learning Model as ".sav" file - `pip install Joblib`
     - **LazyPredict** - For Model Comparision - `pip install LazyPredict`
     - **scikit-learn** - For Machine Learning Algorithms - `pip install scikit-learn`
     - **djangorestframework** - For creating APIs and JSONParser - `pip install djangorestframework`
     - **corsheaders** - For connecting Frontend and Backend and allow other to websites access the application - `pip install corsheaders`
